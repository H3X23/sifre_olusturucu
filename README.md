# 🔐 H3X Password Generator

Bu basit Python aracı, rastgele sayılar ve harfler kullanarak güvenli ve tahmin edilmesi zor şifreler oluşturur.

## 🚀 Özellikler
* Rastgele tam sayılar üretir.
* 8 karakterli rastgele harf dizileri (büyük/küçük karışık) oluşturur.
* Oluşturulan şifreleri ekrana yazdırır.
* şifreleri bir `.txt` dosyasına otomatik kaydeder.

## 🛠️ Kullanılan Teknolojiler
* **Dil:** Python 
* **Kütüphaneler:** `random`, `string`

## 📖 Kullanım
Kodu çalıştırmak için terminale veya IDE'nize şu komutu yazın:

```bash
python sifre_olustur.py
